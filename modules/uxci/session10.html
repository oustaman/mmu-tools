Word Count Breakdown

Section 1: ~300 words

Service overview (150-200) + User analysis (150-200)

Section 2: ~900 words

Research (100-150) + Artefacts (300-400) + Principles (200-250) + Testing (150-200) + Iteration (150-200)

Section 3: ~500 words

Problem (200-250) + Solution (200-250) + Recommendations (150-200)

TOTAL: 1,700 words minimum, 2,000 words maximum

What Makes a Good Slide?

✅ Do This:

  • • Clear heading per slide
  • • Scannable (bullets, short paragraphs)
  • • Visual + text balance
  • • Proper word count
  • • Professional look

❌ Avoid This:

  • • Walls of text
  • • No images/visuals
  • • Inconsistent formatting
  • • Tiny unreadable text
  • • Missing slide numbers

Activity: Draft Structure (15 min)

Create slide titles for your entire deck

1. List all slide titles (just headings for now)
2. Estimate word count per slide
3. Check you have all required elements
4. Identify any gaps in content

Peer Review Structures

Swap with a partner and check:

  • ✓ All sections present (1, 2, 3)?
  • ✓ Word counts roughly right?
  • ✓ All required elements included?
  • ✓ Logical flow between slides?
  • ✓ Any obvious gaps?

Common Mistakes

❌ Too many words per slide (unreadable)
❌ No visuals (boring, doesn't show work)
❌ Missing required sections (fails assessment)
❌ No slide numbers (specifically requested!)
❌ Blurry sketch photos (can't evaluate design)
❌ Inconsistent formatting (looks unprofessional)

Slide Numbers REQUIRED

Add slide numbers to EVERY slide

  • 📍Bottom corner works well
  • 🎯Helps with assessment grading
  • ✨Shows professionalism and attention to detail
  • 📋Makes feedback easier to give/receive

Activity: Start Building (20 min)

Open Google Slides or PowerPoint and start building

1. Create all section headers from your structure
2. Copy-paste your written content into slides
3. Add images (sketches, artefacts, flow diagram)
4. Format for readability

Formatting Tips

  • 🎨Consistent fonts: Same font throughout (Arial, Calibri work well)
  • 📏Readable text: Minimum 18pt font size
  • 🌈Good contrast: Dark text on light background (accessibility!)
  • 🎯Professional colors: 2-3 colors max, not too bright
  • 📐Clear hierarchy: Headings, subheadings, body text distinct
