Common Mistakes

‚ùå Missing error states

What if wifi drops? Server error? Already booked? All gone?

‚ùå Dead ends

User reaches screen with no way forward or back

‚ùå Unclear decision points

"User chooses" - but what are the options? Label them!

‚ùå No alternative paths

Only showing the "happy path" - real users don't always follow it

Technical Errors:

  • ‚Ä¢ No internet connection

User Errors:

  • ‚Ä¢ Invalid input
  • ‚Ä¢ Missed required field

Business Logic:

  • ‚Ä¢ Already booked
  • ‚Ä¢ Out of stock
  • ‚Ä¢ Not authorized

Edge Cases:

  • ‚Ä¢ Empty state (no results)
  • ‚Ä¢ First-time user
